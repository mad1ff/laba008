# ОТЧЁТ ПО ЛАБОРАТОРНОЙ РАБОТЕ

## ЗАДАНИЕ 00_distance.py

#### Задание: Составить словарь словарей расстояний между городами.
1. По очереди берем каждый город, сравнивая его с другими городами;
2. Считаем расстояние от него до всех остальных городов по математической формуле: $((x_1 - x_2)^2 + (y_1 - y_2)^2)^{1/2}$;
3. Записываем полученные числа;
4. Проделываем то же самое для остальных городов;
5. Выводим словарь.
