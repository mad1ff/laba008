## ЗАДАНИЕ 00_distance.py

#### Задание: Составить словарь словарей расстояний между городами.
1. По очереди берем каждый город, сравнивая его с другими городами;
2. Считаем расстояние от него до всех остальных городов по математической формуле: $((x_1 - x_2)^2 + (y_1 - y_2)^2)^{1/2}$;
3. Записываем полученные числа;
4. Проделываем то же самое для остальных городов;
5. Выводим словарь.

## ЗАДАНИЕ 01_circle.py

#### Задание: Рассчитать площадь круга, определить, находятся ли точки в круге.
1. Рассчитать по формуле - $`S=π*R^2`$ - площадь круга;
2. Вывести на консоль значение площади этого круга (с точностью до четырёх знаков после запятой);
3. Рассчитать по теореме Пифагора расстояние от середины круга до точки: $\sqrt{x^2 + y^2}$
4. Вывести на консоль True, если точка point_1 лежит внутри круга, и False, если она не лежит;
5. То же самое проделываем с другой точкой - point_2.
